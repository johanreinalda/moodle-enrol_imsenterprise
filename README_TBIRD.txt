Moodle IMS Enterprise Enrolment module with
Thunderbird School of Global Management customizations

INSTALLATION

Please see INSTALL.txt.

DESCRIPTION

This enrolment module is a customization of the original Moodle 2.x IMS
enrolment module. It adds snapshot unenrolment, and several other options.
It also add the ability to add end-dates to courses, but it needs a database table for this.
These arec currently NOT installed by the module install routines. You will need
to create these by hand in the database. See README_DB_TABLES.TXT for more.
